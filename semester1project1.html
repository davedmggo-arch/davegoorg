Project 1: 3D Fundamentals (Weeks 4–7)

You will build a simple 3D scene composed of a table and props and a simple character made from primitive shapes. This project introduces basic 3D modeling tools, shape language, silhouette design, and scene composition. You will also apply materials, light and render your scene from multiple camera angles.

Requirements

  • A table with at least the three props that were designed in the last project on a ground plane. Include a chair, if it compliments the scene
  • A simple 3D character constructed from primitive shapes (cube, sphere, cylinder, etc.).
  • Three rendered views (different angles).

Character and Table Choices (Assigned by Theme)

  • Woodland Kitchen: Lumberjack, Woodland Creature, Chef with a Kitchen Table, Farmhouse Dining Table, Drop Leaf Table
  • Prince’s Bedroom: Servant, Prince, Thief with a Vanity Table, Nightstand, Study Desk
  • 1900s Workspace: Blacksmith, Explorer, Architect with a Drafting Table, Workbench, Console Table
  • Cozy Living Room: Student, Couch Potato, Gardener on a Coffee Table, Center Table, End Table

Deliverables:

Images of three views of the table and props with the character beside it.
  • Rendered as 1920 x 1090 JPG images with the following naming convention
  • FirstName_LastName_Project_1_ANI106_v01_01.jpg
  • If you re-submit, increment the version number (v02, v03…).

One Blender file
Proper layer organization (different objects in different collections, consistent naming convention).
  • Inside each prop object collection, have proper names of the different components, have proper names for the different materials, make sure that the scale of each object is correct to similar, real world objects and the XYZ scales are set to 1
  • FirstName_LastName_Project_1_ANI106_v01.blend
  • If you re-submit, increment the version number (v02, v03…).

Please combine all files in a .zip folder named FirstName_LastName_Project_1_ANI106_v01.zip and submit it to this project.

Guidelines

  • Always consider form vs functionality when creating objects
  • Keep in mind strong silhouettes, use of the varying degrees of detail, and establish a basic colour scheme to set the mood of your set design
  • Focus on consistency between objects (objects need to be used by the same character, handles and diameters of shapes must suit the needs of the scene).
  • Think about design unity within your theme (e.g., a rustic wooden lantern vs. a futuristic glowing lantern).
