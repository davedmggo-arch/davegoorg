Project 1: Designing and Prototyping a Robot

Weeks 1-3: Design and prototype a stylized robot character from scratch, learning the basics of 3D design, shape language, and Blender navigation.

View Project Details
Back to Semester 1 Week 1

Project Brief

Overview

In this project, you will design and create a stylized robot character from scratch using Blender 4.4. This foundational project focuses on teaching you the essential skills of 3D design, shape language, and Blender navigation without requiring any prior 3D experience.

Project Goals

  • Learn Blender 4.4 interface and navigation fundamentals
  • Master Grease Pencil tools for 2D sketching and concept development
  • Understand and apply visual design principles (silhouette, shape hierarchy, proportions)
  • Create orthographic drawings for 3D production preparation
  • Translate 2D concepts into 3D using Clay Pencil geometry nodes
  • Apply basic materials and color blocking for visual clarity

Deliverables

  • Week 1: 4 robot silhouette sketches in Grease Pencil
  • Week 2: 2 robot orthographic drawings (front + side views)
  • Week 3: 1 stylized 3D Clay Pencil model with base materials

Learning Outcomes

By the end of this project, you will have developed a strong foundation in 3D design principles, Blender workflow, and the ability to translate creative concepts from 2D to 3D. You'll understand how to create visually appealing robot designs that communicate function and personality through shape and form.

Marking Criteria

Assessment Breakdown

30%

Design Fundamentals

Evaluation of silhouette strength, shape hierarchy, proportions, and function-based design principles. Clear visual communication through shape language and consistent design logic across all views.

25%

Technical Execution

Proficiency in Blender 4.4 interface navigation, Grease Pencil tool usage, orthographic drawing accuracy, and Clay Pencil geometry node application. Clean layer organization and proper file management.

25%

Creative Problem Solving

Innovation in robot design, effective translation of 2D concepts to 3D form, thoughtful material choices, and ability to iterate and refine designs based on feedback and design principles.

20%

Process & Documentation

Quality of sketch development, documentation of design decisions, proper file naming conventions, and adherence to project timeline and deliverable requirements.
