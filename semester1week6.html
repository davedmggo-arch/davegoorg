Character Block-Out
Set the goal: stylized 4â€“6 heads tall; gather 1â€“2 quick refs.

Place primitives:
  • Head: sphere.
  • Torso: cylinder or box.
  • Arms/Legs: cylinders.
  • Joints: spheres (shoulders, elbows, hips, knees).

Proportion pass: check widths/heights from front/side; adjust before details.
Pose blocking: rotate/place limbs so the character uses or reacts to the table (reach, lean, look). Keep a clear line of action.
Readability checks: keep forms simple, test silhouettes often, and rely on reference to avoid drift.
Create a Clean Material Library
Naming rules: lowercase, no spaces, use underscores; descriptive types: wood_oak, metal_chrome, ceramic_white.

Principled BSDF quick refresher: Base Color, Metallic (0=non-metal like wood/fabric; 1=metal), Roughness (0 mirrorâ€¦1 matte), Specular (reflection intensity), Transmission (glass).

Make these scene materials with sensible defaults:
  • wood_oak: Base warm brown; Metallic 0; Roughness 0.6.
  • metal_chrome: Base light gray; Metallic 1; Roughness 0.1.
  • ceramic_white: Base white; Metallic 0; Roughness 0.2.
  • glass_clear: Base very light; Metallic 0; Roughness 0; Transmission 0.9.
  • porcelain_white (vase): Base white; Metallic 0; Roughness 0.1.

Scene Material Integration & Harmony
Run this checklist (iterate until cohesive):

  • Material contrast: vary roughness; mix metal/non-metal; create focal points.
  • Environmental consistency: materials fit the mood & lighting.
