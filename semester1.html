Class Outcomes:

By the end of this semester, students will:
  • Master Blender's interface and fundamental digital art principles including file organization and technical drawing.
  • Demonstrate understanding of 3D space, coordinate systems, and basic modeling using primitive shapes.
  • Apply shape language, silhouette design principles, and subdivision surface modeling techniques.
  • Create basic materials and demonstrate understanding of modular environment design in Blender.

Tips & Tricks for the Semester:

  • Save frequently and use version control: Always work at higher resolution than needed for final output and use consistent file naming (ProjectName_Asset_Version_Date).
  • Master navigation shortcuts: Middle mouse button for orbit, Shift + Middle mouse for pan, scroll wheel for zoom, and numpad keys for quick view changes.
  • Use layers non-destructively: Keep original sketches on separate layers and maintain organized folder structures for all projects.
  • Practice fundamentals daily: Regular practice with basic shapes, navigation, and interface tools will build muscle memory and speed up your workflow.

PROJECT 0: GREASE PENCIL FUNDAMENTALS (Weeks 1-3)

Students will create a set of technical drawings demonstrating proficiency with Blender's Grease Pencil tools and interface navigation, including multiple views of a simple object with consistent proportions, proper layer organization, and documentation of their workflow progression.

PROJECT 1: 3D FUNDAMENTALS (Weeks 4-7)

Students will create a simple 3D scene featuring a character and environment using primitive shapes and basic 3D modeling techniques, demonstrating their understanding of shape language, silhouette design, and basic scene composition with professional lighting and multiple rendered views.

PROJECT 2: FUNDAMENTAL ENVIRONMENT DESIGN (Weeks 8-12)

Students will create a complete 3D environment using modular design principles and subdivision surface modeling, developing at least three modular pieces and two detailed props with complete material setups, professional lighting, and technical documentation.

Time Commitment & Expectations

  • Weekly Commitment: 3 hours in-class + 3 hours homework
  • Project 0 (Weeks 1-3): 18 hours total (20% of semester grade)
  • Project 1 (Weeks 4-7): 24 hours total (30% of semester grade)
  • Project 2 (Weeks 8-12): 30 hours total (50% of semester grade)
  • Total Semester Hours: 72 hours
