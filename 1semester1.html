3D Integration Semester 1

This curriculum is designed for animation-arts students to master the core 3D workflow in Blender, covering everything from hard-surface modeling to character rigging and interior design.

Start Learning!!

Project Briefs

Project 1 (Table & Props)

You will build a cleanly modeled table with three distinct props, UV map them, and present the final scene using professional three-point lighting.

Project 2 (Basic Character)

This project focuses on creating a character bust by sculpting a head, torso, and arms, then retopologizing and rigging it for a simple animation test.

Project 3 (Interior Design)

You will design a functional interior "acting space" using procedural materials, multiple camera angles, and advanced interior lighting techniques.

Project 4 (Advanced Character)

You will create a full-body character through high-fidelity sculpting, manual retopology, and complex multi-channel material layering.
