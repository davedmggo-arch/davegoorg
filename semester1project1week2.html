Week 2: Grease Pencil Tools & Technical Drawing

This week focuses on mastering Grease Pencil’s drawing tools, organizing layers, and producing precise technical drawings. Students will create multi-view orthographic projections (front, side, top) of a simple object, applying correct line weights, proportions, and materials. The goal is to bridge freehand drawing with technical accuracy, preparing students for 3D modeling.

Workspace Setup
  • Switch to Drawing Workspace: Select “Drawing” at top, or create a new one.
  • Add a Grease Pencil Object: Shift+A → Grease Pencil → Blank.

Layer Setup:
  • 01_Construction
  • 02_Outlines
  • 03_Details
  • 04_Annotations
  • Lock/hide as needed.

Practice each tool:
  • D = freehand sketch lines.
  • E = erase strokes.
Creating & Applying Materials
Go to Properties → Materials tab.
Click + to add a new material.

Create these starter materials:
  • Ink_Black (Stroke on, Fill off).
  • Hidden_Dash (Stroke dashed, lighter color).
  • Construction_Light (Light blue stroke, thin line).
  • Fill_Flat (Fill only, solid color).

Assign materials to different layers: outlines = Ink_Black, construction = Construction_Light, etc.
Test filling shapes with F; if leaks occur, increase Fill Threshold.
Technical Drawing Fundamentals Drill
Object choice: Cup, chair, or lamp.

Front View (on 01_Construction):
  • Draw bounding box (width × height).
  • Block in key shapes (e.g., ellipse for cup rim, box for chair seat).

Project to Side View (new layer):
  • Use grid snapping (magnet icon → Increment).
  • Project horizontal guides to maintain height/width consistency.
  • Add depth dimensions (chair legs extending, lamp body thickness).

Project to Top View (new layer):
  • Align horizontally above front view.
  • Project lines downward to align with front and side widths.
  • Draw ellipses for circular forms, rectangles for square forms.

Refine Outlines (02_Outlines):
  • Use heavier strokes for visible edges.
  • Switch to Hidden_Dash for hidden lines.
  • Keep construction lines on a separate locked layer.

Add Details (03_Details):
  • Functional details (handle, hinge, lampshade edge).
  • Ensure details align across all views.

Annotations (04_Dimensions):
  • Add arrows or labels for key dimensions.
  • Keep annotations thinner than outlines.
Multi-View Alignment Check
  • Ensure front, side, and top views are aligned on grid.
  • Double-check all proportions (heights/widths/depths match across views).
  • Hide construction lines to test final readability.
  • Export test render as PNG for feedback.
