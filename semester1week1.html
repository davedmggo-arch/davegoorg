Getting Started
Update drivers
  • Install the latest Wacom/Huion/XP-PEN driver. Reboot if prompted.

OS-level pen settings
  • Windows 10/11: In vendor app, keep Windows Ink ON (default).
  • macOS: No special toggle; grant tablet app Accessibility/Input Monitoring if asked.

Blender input settings
  • Edit ▸ Preferences ▸ Input → Tablet:
  • Windows: set Tablet API = Windows Ink (try Wintab only if Ink misbehaves).
  • macOS/Linux: Tablet API = Automatic.
  • (Optional) Edit ▸ Preferences ▸ Navigation → enable Emulate 3-Button Mouse if your pen lacks a middle-click.

Pen button mapping (in tablet driver)
  • Tip = Left-click (draw).
  • Lower side button = Right-click (menus).
  • Upper side button = Middle-click (orbit/pan with modifiers).

Pressure verification inside Blender
  • Add ▸ Grease Pencil ▸ Blank.
  • Select Draw tool. In Tool Settings ▸ Brush, enable Pressure for Radius and Strength.
  • On a new layer “PressureTest”, draw a few lines from feather-light to heavy in a single stroke; confirm taper and opacity respond.

If pressure fails
  • Unplug/replug tablet after Blender is open; toggle Windows Ink in driver; switch Blender Tablet API to the other option and re-test.

Open Blender for 2D Animation

  • File → New → 2D Animation

  • You'll now see a blank canvas workspace.

Customizing Shortcuts

  • Why Shortcuts Matter

    • Faster switching between tools.

    • Builds workflow habits early.

  • Assign Shortcuts

    • Right-click Draw Tool → Assign Shortcut → press D.

    • Right-click Erase Tool → Assign Shortcut → press E.

    • D = Draw and E = Erase, are simple, memorable setup for all future lessons.

Navigating the Workspace
Blender is a 3D Program
  • Aside from 2D view, Blender includes a 3D grid and depth.
  • For the first 3 weeks, we’ll stay focused on 2D drawing.

Basic Controls
  • Rotate View: Alt + Click + Drag → see 3D grid (just to understand space).
  • Reset View: Click Camera icon (right toolbar) → back to 2D canvas.
  • Pan: Alt + Shift + Drag.
  • Zoom: Alt + Ctrl + Drag.

Artistic Tip
  • Zoom out often to check composition and proportions.
  • Great art requires seeing both details and the whole canvas.
First Drawing Exercise — Organic Shapes

Freehand vs. Perfect Shapes

  • Blender’s shape tools make perfect squares/circles.

  • These are useful, but they often look static and lifeless.

  • Freehand lines → imperfections give character, rhythm, and life.


Practical Exercise

  • Draw a square by hand using the Draw Tool.

  • Apply heavier pressure on one side → creates weight, depth, and dimension.

  • Compare with a computer-generated square → freehand feels more alive.


Learning to control your digital strokes:
Radius (size):
  • Adjust via the radius slider at the top of the screen.
  • Or press F, then drag to change brush size interactively. 

Strength (opacity/intensity):
  • Change via the strength slider in the toolbar.
  • Or press Shift + F, then drag to adjust strength interactively. 
Week 1 Practice Assignment

Shapes to Practice

  • Square

  • Circle

  • Triangle

  • Star

  • Heart


Guidelines

  • Draw freehand, no shape tool.

  • Focus on confidence, not perfection.

  • Use line weight to add depth and dimension.

  • Repeat each shape multiple times until comfortable.

File Organization & Save Discipline
  • Folder tree: Semester1/Project0/Week01/ with subfolders /BLEND/, /RND/, /DOC/.
  • Name: P0_Week01_Shapes_YourName_v01.blend; bump v02, v03… for milestones.
  • Save every 10 min; keep raw .blend separate from exported PNG/JPG; use cloud or external backup.

Artistic Mindset & Growth

Anyone can get better at drawing with time and practice.

Improvement comes from:
  • Understanding your strengths.
  • Identifying your weaknesses.
  • Developing both for balance.

Knowledge in all forms of art makes you more versatile and whole as a creator.
