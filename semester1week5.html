Week 5: From Orthographic References to a Clean Blockout

In this lesson, you will learn how to take orthographic reference images and bring them into Blender so you can build accurate models. You’ll start by blocking out your design with simple primitive shapes, then refine it in Edit Mode using tools like loop cuts, bevels, insets, and extrusions in the same way you build your tables. By the end, you’ll understand how to keep your geometry clean, use modifiers to speed up your workflow, and present your model clearly with organized files and viewport renders.

In-Class Topics

  • You will learn how to evaluate and prepare good orthographic reference images for modeling.
  • Import and correctly align front, side, and top reference views into Blender at scale.
  • Block out primary forms using primitive meshes, keeping proportions consistent with references.
  • Refine geometry in Edit Mode using essential tools like loop cuts, bevels, insets, and extrusions.
  • Apply and manage modifiers non-destructively, clean up their models, and prepare clear viewport renders.

Key Concepts & Tools


Orthographic References
  • Consistency: All views must show the same design.
  • True orthographic: Avoid perspective distortion when capturing or drawing references.
  • Centerlines and scale: Use a centerline and at least one real-world dimension for accuracy.
  • Cleanup: High contrast, minimal shadows, exported as PNGs.
Scene Setup & Importing
  • Units: 1 Blender unit = 1 meter (industry standard for scale).
  • Collections for Organization: Keep reference images separate from meshes.
  • Import Reference Images: Use Add > Image > Reference while in the correct orthographic view.
  • Aligning References: Match scale by measuring a known dimension and adjusting the image height/width.
  • Locking References: Prevents accidental selection during modeling.

Tips & Tricks
  • If refs don’t line up perfectly, prioritize the front view. Your eyes will be trained mostly on silhouette.
  • Use transparency sliders (opacity ~0.5) to see geometry through the images.
Primary Forms with Primitives
  • Start with Simple Shapes: Cube, cylinder, sphere, torus. Don’t overcomplicate early.
  • Match Big Proportions First: Get height, width, and depth correct before adding curves or details.
  • Mirror Symmetry: Place half an object and mirror across centerline to save time and ensure balance.
  • Apply Scale/Rotation: Use Ctrl+A after reshaping primitives so tools behave predictably later.
  • Hierarchy of Forms: Primary (overall volume) → secondary (major features) → tertiary (small details).

Tips & Tricks
  • Don’t “carve” everything from one cube. Add multiple primitives for clarity.
  • Use wireframe mode (Shift+Z) to check alignment against refs.
  • Name objects as you go (Body, Handle, Legs) — avoids messy Outliner later.
Edit Mode for Secondary Forms
  • Loop Cuts (Ctrl+R): Add more edges for curvature or support.
  • Bevels (Ctrl+B): Round off sharp edges and add realism.
  • Inset (I): Great for lips, borders, or recessed panels.
  • Extrude (E): Build geometry outward/inward to form handles, ridges, etc.
  • Edge Slide (GG): Adjust loops without distorting mesh.
  • Topology Basics: Favor quads (4-sided faces); avoid messy triangles/n-gons on curved areas.

Tips & Tricks
  • Small bevels = realism. Nothing in the real world is a perfect sharp corner.
  • Use the bottom-left “Adjust Last Operation” menu for precision after loop cuts or bevels.
  • Don’t add too much detail too soon. Keep edge flow even and density manageable.
Modifiers & Non-Destructive Workflow
What Modifiers Are
  • Automatic tools in Blender that modify geometry non-destructively.
  • Think of them as “rules” or “filters” you can stack on your model.
  • You can turn them on/off, change settings, or apply them permanently.

Why Use Them
  • Save time, keep models editable, reduce repetitive work.

Common Modifiers for Beginners
  • Mirror Modifier: Duplicates one half across a centerline and ensures symmetry.
  • Solidify Modifier: Adds thickness to thin objects (like walls or a mug lip).
  • Bevel Modifier: Automatically rounds selected edges.
  • Subdivision Surface Modifier: Smooths geometry by subdividing faces.

Tips & Tricks
  • Always keep modifiers in the Modifier Stack (right-side panel) unless you need to apply them.
  • Start with Mirror. Most objects are symmetrical.
  • Keep Clipping enabled on Mirror to avoid a seam gap.
  • If a modifier “breaks” the mesh, check if transforms are applied (Ctrl+A).
  • Don’t use Subdivision Surface as a shortcut to fix messy base geometry. Clean forms first.

Blender Shortcuts

Navigation & Views
  • Numpad 1/3/7 = Front/Right/Top view
  • Ctrl + (1/3/7) = Opposite views (Back/Left/Bottom)
  • Numpad 5 = Toggle Perspective/Orthographic
  • ~ (Tilde) = View pie menu
  • Shift+Z =Wireframe mode

Selection
  • 1/2/3 = Vertex/Edge/Face select (Edit Mode)
  • Alt+Click = Select edge/face loop
  • Ctrl+L = Select linked geometry
  • A = Select all / Alt+A = Deselect all

Transform & Snapping
  • G = Grab/Move
  • R = Rotate
  • S = Scale
  • Constrain with X/Y/Z or exclude with Shift+X/Y/Z
  • Ctrl+A = Apply transforms (Rotation/Scale)
  • Shift+Tab = Toggle snapping
  • Shift+S = Snap pie menu

Edit Mode Tools
  • Ctrl+R = Loop cut
  • Ctrl+B = Bevel edges
  • I = Inset faces
  • E = Extrude
  • Alt+E = Extrude along normals
  • GG = Edge slide
  • Alt+S = Shrink/Fatten
  • K = Knife tool
  • M = Merge vertices
  • F = Create face
