Week 1: Foundations & The Blender Interface

You will demystify the Blender interface and master primary mesh editing tools to begin the structural "blocking" of your table model.

Start Learning!!

Learning Objectives

  • Master interface navigation and primary mesh editing tools.
  • Understand the 3D Cartesian coordinate system: X (Red), Y (Green), and Z (Blue).
  • Demystify the 3D Viewport to fluidly stage objects and inspect geometry.

Key Concepts

1. Downloading & Installing Blender

  • Official Source: Download only from blender.org to ensure you have the official, safe version.
  • Version Choice: Use the Latest Stable version for new features or the LTS (Long Term Support) version if you prioritize maximum stability for long projects.
  • System Requirements: A minimum of 8GB RAM (16GB recommended) and a dedicated graphics card (GPU) with at least 2GB of VRAM are ideal for a smooth experience.
  • Installation: Follow the standard installation wizard for Windows or drag the application to your Applications folder on macOS.

2. Optimal Configuration for Learning

  • Splash Screen: On your first launch, keep the "New Setup" defaults. Ensure your "Select With" is set to Left Click.
  • GPU Acceleration: Navigate to Edit > Preferences > System. Under Cycles Render Devices, select Optix or CUDA (NVIDIA) or HIP (AMD) to use your graphics card for faster rendering.
  • Laptop Users: If you don't have a number pad, go to Preferences > Input and check Emulate Numpad. This lets you use the 1-0 keys for camera views.
  • Essential Add-on: In Preferences > Add-ons, search for and enable Node Wrangler. This is a vital tool for the texturing work we will do in later weeks.
  • Interface Scale: If the text is too small for your monitor, adjust the Resolution Scale under Preferences > Interface.

3. Interface & Navigation

  • 3D Viewport: This is your main stage where you interact with your 3D models.
  • The Outliner: Located at the top right, this acts like a "Layers" panel, tracking every object in your scene.
  • Properties Panel: Located at the bottom right, this is where you modify object data, materials, and settings.
  • Navigation Shortcuts:
    • Rotate: Middle Mouse Button (MMB).
    • Pan: Shift + MMB.
    • Zoom: Scroll Wheel.
    • Focus: Numpad Period (.) to center your view on a selected object.

4. Object Mode vs. Edit Mode

  • Object Mode: Used for global changes. This is where you manage the "containers" of your geometry.
    • Add Object: Shift + A to open the Add menu.
    • Delete: X or Delete key.
    • Transformation: G (Grab/Move), R (Rotate), and S (Scale).
    • Constrain by Axis: After pressing G, R, or S, press X, Y, or Z to lock movement to a specific axis.
    • Numerical Input: Type numbers for precision (e.g., G + Z + 2 moves an object exactly 2 meters up).
    • Duplicate (Standard): Shift + D creates an independent copy.
    • Duplicate (Linked): Alt + D creates an instance where editing the original changes all copies.
  • Edit Mode: Used for local changes where you manipulate the specific geometry (vertices, edges, and faces).
  • Toggle: Use the Tab key to switch quickly between these modes.

5. Edit Mode & Selection Modes

Note: The following tools and selection modes will only work once you have selected an object and toggled into Edit Mode (Tab).

  • Selection Modes: Use the number row keys to choose what part of the geometry you are touching:
    • 1: Vertex Selection – Grab individual points.
    • 2: Edge Selection – Grab the lines connecting points.
    • 3: Face Selection – Grab the flat surfaces.
  • Advanced Selection Methods:
    • Select Loop (Alt + Left Click): Selects an entire continuous ring of edges or faces. This is the fastest way to select the circumference of a table leg.
    • Select Linked (L): Hover your mouse over a piece of geometry and press L to select everything physically connected to that piece. This is great for grabbing a single prop without touching the rest of the scene.
  • Primary Modeling Tools:
    • Extrude (E): Pulls new geometry out of a face to "grow" shapes.
    • Loop Cut (Ctrl + R): Adds a ring of edges to define new detail areas.
    • Bevel (Ctrl + B): Rounds off sharp edges to catch light and add realism.
    • Inset (I): Creates a new face inside an existing face, scaled inward.

6. Understanding Clean Topology

  • Quads are King: Strive for four-sided polygons to ensure predictable deformation during animation.
  • Avoid N-Gons: Faces with five or more sides cause "shading artifacts" and irregular shadows.
  • Consistent Face Size: Keep your polygons roughly the same size across the mesh to avoid stretching during UV unwrapping.
  • Follow the Form: Align your edge loops with the natural contours or muscle lines of the object to support clean bending.
  • Avoid "Poles": Try to prevent more than five edges from meeting at a single vertex, as this can cause pinching in the mesh surface.

7. Advanced Viewing & Interface Tools

  • X-Ray Mode (Alt + Z): Makes objects semi-transparent so you can select vertices on the back side of a model.
  • Wireframe Mode (Shift + Z): Toggles between Solid view and a see-through wireframe view.
  • Orthographic Views: Use the Numpad (1: Front, 3: Side, 7: Top) for flat, perfectly accurate 2D perspectives.
  • Laptop Hotkeys: If Emulate Numpad is on, use the top row numbers (1-0) to switch these views.
  • The N-Panel: Press N to open the sidebar for Transform data, Tool settings, and View options.
  • The Left Toolbar: Press T to toggle the visual icons for basic tools like Select, Move, and Rotate.

8. Week 1 Assignment: The Table Project

  • The Task: Create a structurally sound, detailed table using a blockout-to-detail workflow.
  • Step 1: The Blockout (Object Mode):
    • Create the Table Top using a Cube (Shift + A), then Scale (S) it to size.
    • Create one Table Leg. Apply Transformations (Ctrl + A > Scale) to ensure the math is reset to 1.0.
    • Use Linked Duplicates (Alt + D) to create the other three legs. This ensures that detailing one leg later will update the others automatically.
  • Step 2: Detailing (Edit Mode):
    • Select the Table Top, enter Edit Mode (Tab), and add Bevels (Ctrl + B) to the edges.
    • Use Insets (I) and Extrusions (E) to add recessed panels or frames to the underside.
    • Select one Table Leg and add Loop Cuts (Ctrl + R) to define areas for decorative tapering or support beams.
  • Submit: A screenshot of your completed table mesh in Solid Mode and a second screenshot in Wireframe Mode.

9. Modeling Best Practices

  • Block Out First: Always establish the large shapes and overall scale before diving into small details like bevels or decorative parts.
  • Be Aware of Scale: Model to real-world dimensions. A standard dinner table for 4 people is typically 1.2m to 1.5m in length, 0.75m to 0.9m in width, and 0.75m in height.
  • Link Duplicates: Use Alt + D for repetitive parts (legs, bolts, chairs) to save memory and speed up future editing.
  • Apply Scale Regularly: Before using tools like Bevel or UV Unwrapping, use Ctrl + A > Scale. If your scale isn't 1.0, your tools will behave unpredictably.
  • Name Your Objects: Double-click objects in the Outliner to name them (e.g., "Table_Top"). It prevents confusion as your scene gets more complex.
  • Work Small to Large: Fix any structural or topology issues at the lowest resolution possible before adding more loop cuts or subdivisions.
