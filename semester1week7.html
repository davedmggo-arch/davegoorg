In-Class Topics

Environment design principles: focal points, scale/perspective for depth, mood/atmosphere, “show don’t tell.”
Lighting basics: three-point setup (key/fill/rim), light types and naming conventions, color for mood.
Camera setup & composition: camera types/settings, rule of thirds, angle choices, switching and testing.
Render prep & submission standards: multiple views, wireframe/material screenshots, organized package.

Environment Storytelling Enhancement
  • Pick a story beat (cozy breakfast, late-night study, repair bench). Keep it visual—no text. Show, don’t tell; context is key; details matter.
  • Establish scale & depth: add a simple wall/floor or window plane; layer props into foreground/midground/background; vary sizes; consider forced perspective for drama.
  • Mood & atmosphere: select a color palette (warm = cozy, cool = distant) and optional light fog/dust for depth; match time-of-day.
  • Lock focal point(s): decide where the eye should land (the table scene), then place supporting details that guide attention. Use leading lines and thirds.
Three-Point Lighting Setup
  • Key: Shift+A ▸ Light ▸ Area → place ~45° to subject; rotate toward scene; start around 500 W, warm white. Name light_key.
  • Fill: Add Area on the opposite side; ~250 W (≈½ key), slightly cooler. Name light_fill.
  • Rim: Add Area behind/above; ~200 W, neutral white to outline forms. Name light_rim.
  • Tune mood: nudge colors/intensities to fit the story; ensure key forms are readable and shadows not crushing detail.
Cameras & Composition
Main camera: Shift+A ▸ Camera → name cam_main; frame the whole scene; start 50 mm focal length for natural perspective.
Detail camera: add cam_detail; frame character–table interaction; try 85 mm for compression.
Overhead: add cam_overhead; place above; set Orthographic for plan-like clarity.
Composition pass: apply rule of thirds, leading lines, and layering for depth; test each camera and adjust.
Switch & review: select a camera → Ctrl+Numpad 0; bookmark favorites and iterate.
Render Prep & Quality Check
  • Scene check: materials consistent and named; elements refined; silhouettes still strong.
  • Render settings: optimize for quality (samples/resolution) and stability; prep multiple views.
  • Extra outputs: produce wireframe and material preview screenshots for documentation.
Submission Package

Use the dedicated time window to finish, document, and upload. Include: finalized scene/renders, process notes, and clean file organization.
