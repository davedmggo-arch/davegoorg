Week 4: 3D Space Fundamentals & Table Composition

Students transition from 2D Grease Pencil work to 3D modeling by learning how to navigate Blender’s 3D space and coordinate systems. They create a table and basic props using primitive shapes, practice object transformations, and explore silhouette design for readability. Navigation drills ensure fluency in orbiting, panning, zooming, and switching views.

Key Concepts & Tools

Coordinate Systems Drill
  • Spawn a reference cube (Shift+A → Mesh → Cube). Grab with G, then press X, Y, Z to feel global constraints; note the red/green/blue gizmo axes.
  • Rotate the cube (R then axis). Move again with G, press the same axis key twice (e.g., X→X) to switch to local space; observe how the move aligns to the object’s new orientation.
  • Takeaway: You’ll use global for layout and local for part-wise adjustments.
Build the Table from Primitives
Tabletop
  • Shift+A → Mesh → Cube.
  • S, Shift+Z to flatten/scale only in X/Y; G, Z lifts it to table height.

Leg 1
  • Shift+A → Mesh → Cylinder.
  • S to thin (X and Y), S, Z to make tall; G to position at one corner.

Duplicate legs
  • With leg selected, Shift+D then move to each remaining corner.

Optional beams (stability/detail)
  • Add Cube → scale long/thin; position between two legs; Shift+D to mirror around the frame.

Join
  • Select all parts, Ctrl+J to form one mesh (keeps the scene clean).

Organization habits
  • Name the object “table_main”; keep future props as separate objects (you’ll join only when appropriate). (Reinforces the “scene organization” part of grading.)
Add Three Props
(Choose any 3 from chair, plate, cup, vase; keep transforms precise and readable.)
  • Chair (block-in): Cube for seat; Cube for backrest (G/Z for height; R for angle); Cylinder legs duplicated via Shift+D.
  • Plate: Cylinder → S, Z to flatten; subtle rim via S (in Edit Mode with top face selected) then G, Z to lift a touch.
  • Cup: Cylinder → S, Z to height; (optional) duplicate for lip; handle as a separate thin torus or a slimmed cylinder bent with transforms.
  • Vase: Cylinder → scale narrow base & wide body using proportional scaling, or add loop cuts and scale rings.

These prop types are explicitly referenced in Week 4 practice and homework; use them to teach proportion + silhouette fast.
Layout, Negative Space & Silhouette Pass
  • Place props on/around the table for a clear read from front/top/right.
  • Silhouette test: switch to Ortho (Numpad 5), check Front/Side/Top (1/3/7); use Solid/Wireframe to verify reads; adjust spacing to avoid tangents/merges.
  • Why it matters: shape language & silhouette are part of Project 1 grading; you’re building that habit now.

Homework

Complete Table Scene — Table + 3 props (chair/plate/cup/vase), proportionally correct with readable silhouettes.
Silhouette Testing — Check multiple angles; write 6–8 sentences on how negative space influenced your placement choices.
Navigation Practice — Capture screenshots from multiple viewpoints and add a short reflection on silhouette principles.
