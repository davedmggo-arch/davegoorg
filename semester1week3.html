Week 3: Animation Interface & Project Completion

This final week of Project 0 ties together Grease Pencil drawing fundamentals with the basics of animation in Blender. Students will learn how to use the Timeline, add and manipulate keyframes, and apply onion skinning to refine motion. They will complete their multi-view technical drawing set (front, side, top) of a chosen object, add a short animation (e.g., a bouncing ball or simple object movement), and prepare a polished submission with organized files and documentation.

Animation Interface Basics
Switch Workspace: Go to Animation or split the viewport → set lower panel to Timeline.

Timeline Navigation:
  • Space = Play/Pause; Left/Right Arrows = Jump frame by frame; Ctrl + Left/Right = Jump to previous/next keyframe.

Add a Grease Pencil Object: Shift+A → Grease Pencil → Blank.

Onion Skinning: In Viewport Overlays, enable Onion Skinning; set range = 1 previous/1 next frame.

Insert Keyframes:
  • Frame 1: Draw circle (ball). Press I → Location.
  • Frame 20: Move ball down (bounce contact). Press I → Location.
  • Frame 40: Move ball up again. Press I → Location.

Playback & Interpolation: Play animation. Open Graph Editor → select Y-Location curve → change handles to Bezier for smoother motion.
Multi-View Drawing Completion
  • Choose Object: Cup, chair, or lamp.
  • Set Layers: 01_Construction, 02_Outlines, 03_Details, 04_Dimensions.
  • Front View: Align bounding box; add outlines and key details.
  • Side View: Project heights/widths from front using grid snap; refine outlines.
  • Top View: Align width/depth from previous views; finalize proportions.

Line Weight Hierarchy:
  • Thick = visible outlines.
  • Dashed medium = hidden lines.
  • Light thin = construction.
  • Medium = dimensions.

Final Pass: Hide construction layer; confirm proportions are consistent across all three views.
File Organization & Submission Prep
  • Folder Tree: Semester1/Project0/Week03/ with /BLEND/, /RENDERS/, /DOC/.
  • File Naming: P0_Final_ObjectName_YourName_v01.blend.

Exports:
  • PNG render of front/side/top views.
  • PNG/JPG of animation frames (optional playblast MP4).

Homework

Final Technical Drawing Set: Complete front, side, and top views of your chosen object; check for proportional consistency.
Final Submission Package:
  • Organized Blender file(s).
  • PNG/JPG renders of drawings and animation.
  • Workflow documentation.
  • Submit by end of Week 3.
