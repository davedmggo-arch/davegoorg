Week 1: Ground Planes, Procedural Paths, and Blocking

The first week of Project 1 is focused on the "Geological and Spatial Foundation." We are not worried about high-resolution details yet. Instead, we are establishing the scale of your world tile and using procedural data to define where life happens versus where the path leads.

Start Learning!!

Key Concepts

1. Terrain Geometry: The Subdivided Foundation

Your environment starts with a single plane, but it requires enough resolution to support physical height and weight paint data. We use Displacement to create the high-level geological features of your chosen theme.

  • Subdivision Logic: Create a 10m x 10m Plane. In Edit Mode, subdivide it at least 50 to 100 times. This resolution is necessary because both Displacement and Weight Painting rely on Vertex Density.
  • Displacement Setup: Add a Displace Modifier. Create a New Texture and set it to Clouds.
    • Forest: Use a large scale for the clouds to create soft, rolling mounds for moss and roots.
    • Shoreline: Use a medium scale with higher strength to create the dips where tide pools will sit.
    • Mountain: Use a small, high-contrast scale to create sharp, jagged base elevations.

2. Procedural Paths: Weight Painting

Instead of painting a texture, we will paint Data. Weight painting allows us to define a "Path" that the material nodes will understand later. This makes your environment non-destructive; you can change the path at any time by simply painting over it.

The Weight Painting Workflow

  • Select your ground plane and switch to Weight Paint Mode.
  • Your mesh will turn blue (0.0 weight). Use the brush to paint red (1.0 weight) where you want your Road to be.
  • Under the Vertex Groups tab in Object Data Properties, rename this group to "Path_Data".
  • Ensure your brush has a soft falloff to allow for natural blending between the path and the surrounding terrain.

3. The Node View: Translating Data to Visuals

Now we connect that data to the Shader Editor. This is the core of proceduralism. We will tell Blender: "Wherever I painted red, show the Road material. Everywhere else, show the Environment material."

  • Open the Shader Editor and create a new material for the plane.
  • The Color Attribute Node: Add a Color Attribute node. Create a "Path" Color Attribute into the Name field. Connect the "Factor" output to the "Fac" input of a Mix Shader.
  • The Mix: Plug a dark, grainy shader (Road) into the bottom input and a textured shader (Grass/Sand/Snow) into the top input.
  • Refinement: Add a ColorRamp between the Attribute node and the Mix Shader to tighten the edges of your path or add "noise" to the transition.

4. Spatial Blocking: Target and Non-Accessible Locations

You will now block in the major focal points using Primitive Placeholders. This is about Composition and Design Principles.

Target Location (The Focal Point)

  • Place this using the Rule of Thirds. If your road leads the eye to the center, it can feel stagnant. Placing it slightly off-center creates dynamic tension.
  • Use a "Large" primitive (e.g., a large cube for a cabin or a tall cylinder for a cliff-side altar).

Non-Accessible Locations (The Boundaries)

  • These objects serve as "Visual Blockers" to keep the viewer contained within the tile.
  • Place these at the edges of the tile. In a Forest, these are "Wall" shapes for dense trees. In a Mountain, these are sheer vertical wedges.

5. Visual Hierarchy in Blocking

Even with simple cubes and spheres, you must demonstrate Form Hierarchy:

  • Large: The main elevation of the Target Location.
  • Medium: Secondary clusters around the target (e.g., smaller rocks or crates).
  • Small: This will be handled in Week 2, but keep space for them in your layout.

Blender Shortcuts for Week 1 Expansion

  • Ctrl + Tab: Quickly switch between Object, Edit, and Weight Paint modes.
  • F: Adjust Weight Paint brush size.
  • Shift + F: Adjust Weight Paint brush strength (opacity).
  • Numpad 7: Top view (Best for painting your road layout).
  • Ctrl + A: Apply All Transforms (Crucial so displacement and weight painting scale correctly).
Week 1 Deliverable Check: By the end of this week, you should have a displaced ground plane, a path defined by weight painting that blends two distinct colors, and a clear block-out of your Target and Boundary locations. Ensure your file naming follows the project guidelines.
