Week 2: Prop Creation & UV Basics

You will populate your scene with three distinct props while learning to "flatten" 3D objects through UV unwrapping to prepare them for textures.

Start Learning!!

Learning Outcomes

  • Identify the difference between standard and linked duplication to manage scene efficiency.
  • Apply the "lathe" and "angular" modeling workflows to create diverse 3D assets.
  • Perform a basic UV unwrap by marking seams and organizing 2D layouts for texturing.

Key Concepts

1. Topology Review: Quality Control

  • Edge Flow: Ensure edges follow a logical path around the model. A clean model allows for selecting continuous rings of edges.
  • Face Counts: Aim for the lowest number of polygons possible that still define the shape clearly. High face counts can slow down your computer and complicate future rigging.
  • Subdivision Readiness: Proper topology ensures that adding a "Subdivision Surface" modifier keeps the model smooth rather than causing "pinching" or distortion.

2. Adding Props: Diverse Asset Creation

This week, you will move beyond the basic box shapes of the table to create three distinct props:

  • Lamp Base (Spin Modeling): Use the Spin Tool to create curved, circular geometry by rotating a profile around a central point.
  • Bookend (Angular Modeling): Focus on rigid 90-degree angles and use "holding edges" (small bevels or extra loop cuts) to maintain sharp corners.
  • Decorative Item (Freeform): Experiment with complex shapes using Inset, Bevel, and Extrude to create intricate patterns or unique silhouettes.

3. Materials & The Principled BSDF

We use the Principled BSDF shader as the "Swiss Army Knife" for all surfaces:

  • Base Color: Set the main pigment or color of your object.
  • Roughness: Controls how light reflects. A value of 0.0 is mirror-like, while 1.0 is matte (like chalk or rubber).
  • Metallic: A simple toggle. Set to 1.0 for metal objects (like a lamp neck) and 0.0 for non-metals (like wood or plastic).

4. UV Unwrapping: Flattening the 3D World

UV Unwrapping is the process of flattening a 3D mesh so a 2D texture can be applied to it.

  • Marking Seams: Think of seams as "cut lines" on a clothing pattern. You tell Blender where to "unzip" the mesh.
  • Unwrapping: Once seams are marked, Blender flattens the geometry into the UV Editor window.
  • Layout Optimization: Organize your "UV islands" (the flat pieces) so they don't overlap and fill as much of the square texture space as possible.

5. Shortcuts & Best Practices

  • Standard Duplicate: Shift + D (Creates an independent copy).
  • Linked Duplicate: Alt + D (Changes made to the mesh of one copy affect all other copies).
  • Mark Seam: Ctrl + E > Mark Seam while in Edit Mode.
  • Unwrap Menu: Press U while all geometry is selected.
  • Apply Scale: Always press Ctrl + A > Scale before unwrapping, or your textures will look stretched.
