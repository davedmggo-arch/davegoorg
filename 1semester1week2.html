Week 2: Prop Creation & UV Basics

You will populate your scene with three distinct props while learning to "flatten" 3D objects through UV unwrapping to prepare them for textures.

Start Learning!!

Learning Outcomes

In this module, we move from the structural "bones" of the table to the "set dressing." We will build three distinct props: a Ceramic Mug, a Desk Lamp, and a Place Mat with a Plate; and prepare them for high-quality image textures.

Key Concepts

A. The Ceramic Mug (Subdivision Modeling)

  • Base Shape: Create a Cylinder (32 vertices). In Edit Mode, delete the top face.
  • Rounding: Select the bottom face and press Ctrl + B to bevel the edge, creating a smooth, rounded base.
  • Solidify: Add a Solidify Modifier from the modifier tab. Set the thickness to 0.4cm so the mug has physical walls instead of being paper-thin.
  • Smoothing: Add a Subdivision Surface Modifier (Level 2). Right-click the object and select Shade Auto Smooth to remove the faceted look.

B. The Plate & Place Mat (Profile & Plane Modeling)

  • The Plate: Start with a Circle. In Edit Mode, use E to extrude and S to scale the vertices inward and slightly upward, tracing the profile of a shallow dish. Apply a Solidify modifier for thickness.
  • The Place Mat: Create a Plane. Scale it to roughly 45cm x 30cm. This represents a standard real-world size, allowing you to check the scale of your table.

C. The Desk Lamp (Hard Surface & Curves)

  • Components: Use a Cylinder for the heavy base and a Cone (with the top vertices deleted) for the shade.
  • The Neck: Use a Bezier Curve (Shift + A > Curve). In the Curve settings panel, go to Geometry > Bevel and increase the Depth. This turns a thin line into a solid, rounded pipe for the lamp's arm.

UV Unwrapping: The Texture Blueprint

Before an image (like wood grain or fabric) can be applied to a 3D object, the object must be "flattened" into 2D space. This process is called UV Unwrapping.

  • Marking Seams: Think of a seam as a "cut" in a piece of fabric. Select edges in Edit Mode and press Ctrl + E > Mark Seam. Place them on the back of the legs or the side of the mug to hide them from the camera.
  • The Unwrap: Select the entire mesh with A, then press U > Unwrap.
  • The UV Editor: Open the UV Editor window to see your 2D "islands." If the shapes look stretched, your texture will look stretched.

Understanding Scale and Image Textures

Realism depends on Texel Density—ensuring the size of the texture matches the size of the object.

  • Applying Scale: Before unwrapping, always press Ctrl + A > Scale in Object Mode. If your scale is not 1.0, your textures will appear distorted.
  • Image Assignment: You will assign specific image files to each object:
    • Table: Wood grain image.
    • Mug/Plate: Smooth ceramic or porcelain image.
    • Mat: Woven fabric or linen image.
    • Lamp: Brushed metal image.
  • Adjusting Scale: If the wood grain looks too large (like a miniature model), use the Mapping Node in the Shader Editor to increase the Scale values until the details look life-sized.
