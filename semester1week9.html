Week 9: Material Fundamentals (Shader Editor, Principled BSDF, UVs & Texturing)

Students learn the Shader Editor (node-based materials), build a small material library with Principled BSDF, then UV unwrap and apply textures to subdivision models. By the end, they’ll create several clean materials, texture at least one prop, and document settings—skills required for the Project 2 final (materials contribute to the Technical Proficiency slice).

In-Class Topics

Shader Editor: opening the editor, node layout, creating/assigning materials.
Principled BSDF properties: Base Color, Roughness, Metallic, Specular, Transmission (+ IOR, Normal).
Texture mapping & UVs: unwrap methods, UV Editor, image textures, coordinate systems.
Material practice: build common materials; apply to models; variations, mixing, and masks.

Key Concepts & Tools

Section content goes here.

Open Shader Editor & Make Your First Material
Open Shader Editor: split the viewport → change new panel to Shader Editor.

Create a Material: select object → Properties ▸ Materials → New → name it clearly. A Principled BSDF and Material Output appear.

Navigate nodes: zoom/pan the node area; connect/disconnect noodles; tweak values live to see viewport updates.
Principled BSDF “Quick Wins”
Use these starter values to understand perceptual ranges (adjust to taste):
  • Plastic: Base Color any; Metallic 0, Roughness ~0.2, Specular ~0.5.
  • Metal (chrome-like): light gray; Metallic 1, Roughness ~0.1.
  • Glass: Transmission 1, Roughness 0, IOR ~1.5.
  • Wood: brown or texture; Metallic 0, Roughness ~0.4.
UV Unwrapping
Enter Edit Mode (Tab) → A (all). Press U → choose Unwrap (start simple).

Open UV Editor (split panel) → confirm UV layout and inspect distortion.

Know your options: Smart UV Project, Lightmap Pack, Follow Active Quads, pick per mesh shape.

Coordinate systems: Generated/Normal/Object/UV (most common for painted textures).
Apply a Texture
In the material, add Image Texture → Open an image → connect Color → Base Color (Principled).

Ensure the model has valid UVs; preview in Material Preview and tweak scale/rotation in UV Editor.
Guided Prop: Stylized Treasure Chest
Model the chest (simple SubD-friendly form, hinges/lock detail, clean topology).

UV unwrap: A → U → Unwrap; fix stretching; arrange islands cleanly.

Wood material: Base brown; Roughness ~0.6, Metallic 0.

Apply textures: Image Texture → Base Color; add roughness/normal maps if available.

Multiple materials: separate wood vs. metal fittings; assign by faces; contrast Roughness/Metallic for readability.
Variations, Mixing & Masks
Mix Shader different looks; drive Factor with textures for wear/dirt zones.

Make material variations (color/roughness tweaks) to avoid repetition; build a mini library.
Tips & Tricks
Check UVs: watch for stretching; keep island scale consistent; pack efficiently.

Dielectric vs. Metal: Metallic=0 for wood/plastic/ceramic/skin; 1 for metals.

Glass needs Transmission and a sensible IOR; keep Roughness near 0 for clarity.

Library mindset: make reusable presets; vary roughness more than color to add realism without noise.

Homework

Repeat all three Exercises: 5 materials documented; one textured SubD prop; a compact library with 2–3 variations per base.
