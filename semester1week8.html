Subdivision Primer
Make it visible: Shift+A ▸ Mesh ▸ Cube → Modifiers → Add Modifier ▸ Subdivision Surface. Raise Levels Viewport to see smoothing; compare with Levels Render.

What’s happening? Catmull-Clark subdivides faces and averages vertices; higher levels = smoother but heavier.

When to use: great for organic curvature and controlled hard-surface forms with support loops.
Edge-Flow Fundamentals
Good flow follows form, deforms predictably, and uses mostly quads; bad flow pinches/stretches and leans on triangles or n-gons.

Plan loops: study reference → place loops along major contours; add only where deformation/detail is needed.

Sanity checks: Edit Mode (Tab) → A → Overlays: Face Orientation; review edge layout for consistency.
Supporting Loops & Edge Creases
Support loops (Ctrl+R): cut near edges that should stay tight; closer = sharper. Even spacing = even curvature.

Edge Crease (Shift+E): select edges → Shift+E → drag/type value; higher = crisper after SubD.

Mini-drill:
  • Add SubD to a cube → all edges round.
  • Add loop cuts around one corner → compare sharpness.
  • Crease a different corner → compare results.

Rule of thumb: use loops first, creases sparingly; keep added geometry minimal.
Guided Build: Stylized Chair
  • Base mesh: Cube for seat; duplicate for backrest; rough proportions first.
  • Control edges: Add Ctrl+R loops where sharpness/support is needed (seat rim, backrest edges).
  • Shape pass: move verts/edges/faces; use Proportional Editing (O) for soft curvature.
  • Crease selectively: Shift+E on edges that must read crisp (e.g., seat perimeter).
  • SubD test: add Subdivision Surface; adjust Levels; iterate shape ↔ loops until clean.
  • Cleanup: check all-quad topology; remove unneeded loops; verify from multiple angles.
  • Pro tips: Apply scale before heavy edits (Ctrl+A ▸ Scale), keep modifiers non-destructive as long as possible, and prefer loops over creases for portable topology (studio-friendly). (General best practice for SubD—reinforces criteria below.)

Homework

Repeat the three Exercises above on fresh meshes; include viewport screenshots at each stage and brief notes on loop placement and crease values.
